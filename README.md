# Claude Code Plugins by jungjaehoon-lifegamez

A curated collection of Claude Code plugins for enhanced productivity and workflow automation.

## Available Plugins

### MAMA - Memory-Augmented MCP Assistant

Track decisions, learn from mistakes, never repeat errors.

An always-on companion that remembers your decision evolution and provides automatic context injection.

**Features:**
- Decision evolution tracking
- Semantic search across your knowledge
- Automatic context injection
- Local-first, privacy-focused
- Multilingual support

**Installation:**
```bash
/plugin marketplace add jungjaehoon-lifegamez/claude-plugins
/plugin install mama
```

[View Documentation](./plugins/mama/README.md)
